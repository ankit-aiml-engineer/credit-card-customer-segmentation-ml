# 💳 Credit Card Customer Segmentation & Prediction System

> 🚀 Machine Learning project to segment customers and enable data-driven decisions
> 💡 End-to-end pipeline with clustering and prediction

---

## 📌 Overview

This project applies machine learning techniques to segment credit card customers based on their financial behavior and interaction patterns.

It helps businesses:

* Understand customer behavior
* Identify high-value users
* Improve marketing strategies

---

## 🎯 Objective

* Segment customers using clustering
* Build prediction model for new customers
* Generate actionable business insights

---

## 📊 Dataset Features

* Avg_Credit_Limit
* Total_Credit_Cards
* Total_visits_bank
* Total_visits_online
* Total_calls_made

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* Streamlit

---

## 🔄 Workflow

1. Data Cleaning
2. Feature Engineering
3. Clustering (K-Means)
4. Model Training (Random Forest)
5. Prediction System
